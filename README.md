# 🐍 Snake HTML5 Plus

Un juego de Snake moderno hecho con HTML5, CSS y JavaScript.

## Características
- Modo Clásico y Modo Plus (con tienda).
- Sistema de monedas y personalización.
- Superpoderes: Guardar/Cargar JSON, Ignorar cola, Velocidad y Tamaño.
- Controles táctiles para móviles.

## Cómo jugar
- Usa las flechas o WASD para moverte.
- Usa los botones táctiles si estás en un celular.
- ¡Gana monedas comiendo y personaliza tu serpiente!
